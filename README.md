# Blog Website

## A blogging platform where you can share your posts with others and can see posts from others.

A blog website made using Next js where you can post your views and share with others. Made to learn about NextJs ans some other technologies like Shadcn and Supabase. It has these features:

- User login and authentication.

- Dark Mode and Light Mode

- Create Blogs which support markdwon

- View Blogs with formatted markdwon

- On this page section which shows contents of blog as well as provide navigation

- Nice Professional ui, have used some shadcn components.

## How to run locally

1. Clone the repository

```
git clone https://github.com/Turbash/anything
cd anything
```

2. Install the dependencies

```
pnpm install
```

3. Run the development surver

```
pnpm run dev
```
